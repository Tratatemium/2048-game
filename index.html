About the game

This project was created as part of my studies in the Front-End Development program at Hyper Island. I recreated Gabriele Cirulli`s original 2048 game↗ as a personal challenge to practice JavaScript algorithms, DOM manipulation, smooth animations, and responsive web design.

How to play

2048 is a sliding puzzle game. Use the arrow keys (or swipe on mobile) to move the tiles. When two tiles with the same number touch, they merge into one. Keep combining numbers to reach the 2048 tile! If you run out of empty spaces, the game is over.
